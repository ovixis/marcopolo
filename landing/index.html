Free & open source · macOS, Windows & Linux · MIT licensed.

One app, the whole trip

Real travel data, charted into a plan

No twenty tabs. Search once; Marco composes flights, stays and experiences into a coherent itinerary and budget.

Aggregated search

Live flights (Duffel), hotels (LiteAPI) and experiences in one place. Real prices, times and names, never invented.

Ask Marco, your agent

An AI travel agent that runs the searches for you and returns a day-by-day itinerary with a clear budget summary.

Local MCP server

Marco runs a local MCP endpoint, so Claude, ChatGPT, Cursor and friends can search travel through the app too.

Bring your own AI

Connect the AI you already use — no API key needed

Marco doesn't lock you to one model. Pick how you want the agent to run.

CLI agent

Claude Code or Kimi Code

Auto-detects the AI CLI you already have installed. Runs on your existing subscription, no key to manage.

No API key
Desktop bridge

Claude Desktop or Kimi

Drives the desktop app you're already signed into. Marco types the question and reads the answer back.

Your subscription
Local model

Ollama, LM Studio, Jan

Auto-detects a local model server already running on your machine. Private, offline, zero key.

Full privacy

Open source, private by design

Your API keys never leave your device. Built to be read, forked and trusted.
